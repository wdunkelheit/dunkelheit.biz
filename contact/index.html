Git

You can contact me regarding an issue with my software by filing an issue on the GitHub or GitLab page, I have no preference. If you do not have an account on these services feel free to email me instead and I will do my best to file an issue on your behalf (anonymously unless stated otherwise) so that a tracker for the issue will exist and I can get around to fixing it.

You can find my Git repos at:

Etiquette

If you are contacting me regarding a GitHub / GitLab issue please include a link to said issue. I get notified when you post the issue on these platforms but sometimes these emails go missing.

If you are contacting me regarding a cryptocurrency transaction make sure to include the relevant hashes and signatures required to verify the authenticity of the claim. I will be publishing a guide at some point on how to verify transaction claims.
